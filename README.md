# Cibo per la Mente

Sito-contenitore di mini-giochi italiani da giocare al volo dal cellulare.
Niente account, niente cookie di tracciamento, niente pubblicità — solo un piccolo divertimento al giorno.

## Giochi disponibili

| Gioco | Descrizione | Cartella |
|-------|-------------|----------|
| **Parola** | Wordle in italiano, 4 livelli di difficoltà (4-7 lettere), 6 tentativi, una parola al giorno per livello | [parola/](parola/) |

## Stack

- HTML + CSS + JavaScript vanilla. Zero build, zero dipendenze npm.
- Hosting: GitHub Pages.
- Analytics: Umami Cloud (anonime, no cookie, GDPR-ok).
- Font: Fraunces + Inter da Google Fonts.

## Per ogni gioco c'è il suo README e la sua specifica funzionale dentro la sua cartella.
